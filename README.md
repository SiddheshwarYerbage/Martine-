🌊 Martine – Travel & Tourism Website

A stylish and responsive Travel & Tourism Web Application built using HTML, CSS, and JavaScript. Martine showcases travel destinations, tour packages, and booking sections with a modern and attractive UI design.

📖 Description
Martine App is a front-end travel website designed to provide users with an engaging experience while exploring different destinations and tour packages.
This project focuses on modern UI layout, responsive design, structured sections, and interactive components that simulate a real-world travel booking platform.
It demonstrates strong skills in front-end development, layout design, and user interface structuring.

🚀 Features
🏖️ Explore featured destinations
📦 Tour package listings
📝 Booking / Contact form
📱 Fully responsive design
🎨 Clean and elegant UI
🔗 Smooth navigation bar

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Media Queries for responsiveness
